# Play With Kubernetes

This repository documents my hands-on learning journey with Kubernetes, following a Senior DevOps Engineer–oriented approach.

## What this repo covers

### Step 1 – Containerizing a Real Application
- Node.js API
- Dockerfile
- Immutable images

### Step 2 – Deployments & Services
- Pods vs Deployments
- Self-healing
- Service networking
- Label-based routing

### Step 3 – ConfigMaps & Secrets
- Externalized configuration
- Secure secret injection
- Runtime configuration

### Step 4 – Health Checks & Autoscaling
- Liveness, Readiness, Startup probes
- Zero-downtime deployments
- Horizontal Pod Autoscaler (HPA)

### Step 5 – Ingress & DNS
- Ingress controllers
- Host & path-based routing
- Debugging 404/502/503 errors

### Step 6 – Persistent Storage
- PV, PVC, StorageClass
- Data persistence across pod restarts
- Stateful vs stateless workloads

### Step 7 – RBAC & Security
- ServiceAccounts
- Roles & RoleBindings
- Least-privilege access control

## Tools Used
- Kubernetes (Minikube)
- Docker
- kubectl
- NGINX Ingress

## Goal
To build a strong, production-ready understanding of Kubernetes from a Senior DevOps Engineer perspective.
